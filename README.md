# Gmail Sender — Инструкция по установке

## Что нужно установить (один раз)

1. **Python** — скачай с https://python.org
   ⚠️ При установке поставь галочку **"Add Python to PATH"**

2. **Зависимости** — открой Командную строку (Win+R → `cmd`) и вставь:
   ```
   pip install customtkinter selenium requests
   ```

---

## Как запустить программу

Положи все файлы в одну папку, затем в cmd:
```
python app.py
```

---

## Файлы в папке

| Файл | Описание |
|---|---|
| `app.py` | Главная программа |
| `profiles.txt` | ID профилей Dolphin Anty (по одному на строку) |
| `templates.json` | Шаблоны писем (создаётся автоматически) |

---

## Как пользоваться

### 1. Профили Dolphin
- Открой Dolphin Anty
- Найди ID нужных профилей (цифры, например `789456`)
- Вставь их в поле **«Профили Dolphin»** — каждый с новой строки
- Нажми **«Сохранить»**

### 2. Получатели
- Скопируй список email-ов из Telegram
- Вставь в поле **«Получатели»**
- Программа сама найдёт все адреса

### 3. Шаблоны писем
- Перейди на вкладку **«Шаблон»**
- Заполни тему и текст
- Нажми **«Сохранить шаблон»**
- Можно создать несколько шаблонов через **«+ Новый шаблон»**

### 4. Рассылка
- Нажми **«▶ Начать рассылку»**
- Прогресс виден на вкладке **«Лог»**
- Можно остановить кнопкой **«⏹ Остановить»**

---

## Важно

- Dolphin Anty должен быть запущен перед стартом рассылки
- Профили чередуются автоматически (3 профиля + 9 писем = 3 письма с каждого)
- Задержку между письмами можно настроить (по умолчанию 5 сек)
